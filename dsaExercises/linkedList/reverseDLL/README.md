
# Problem

Reverse a doubly linked list

[Link](https://www.geeksforgeeks.org/reverse-a-doubly-linked-list/)

## Approaches
1. Brute force
    - Traverse linked list
    - Swap next and previous pointers
    - Change head pointer

2. Stack
    - Traverse linked list
    - Add data values to a stack
    - Unload stack into new DLL
